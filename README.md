# Assignment_5-Scikit-Learn
## Purpose 
THe purpose of this project is to build machine learning classification models using the breast cancer dataset included in Scikit Learn. The project demonstrates the machine learning workflow including loading data, splitting the data into training and testing sets, training models, and evaluting model performance.

## Models Used
Three classification models were used in this project:
- Logisitic Regression
- K-Nearest Neighbors
- Random Forest

## Evalution Metrics
Each model was evaluated using the following metrics:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC AUC

These metrics help determine how well each model performs when predicting whether a tumor is malignant or benign.

The results from the models were compared to determine which algorithm performed best for this dataset.
